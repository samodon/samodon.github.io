WGUPS Parcel Delivery System

Project Date May 18th, 2024
Languages Python
Technologies Hashing Algorithms, K-nearest Neighbor
Link https://github.com/samodon/wgups-delivery-system

The Western Governors University Parcel Service (WGUPS) package delivery simulation project optimizes urban logistics in Salt Lake City, Utah. This system employs a custom-built hash table for efficient package information management and utilizes a k-Nearest Neighbors algorithm to determine the most effective delivery routes. By integrating advanced data structures with machine learning concepts, the simulation creates a robust platform for exploring and enhancing delivery system efficiency. This project not only demonstrates the practical application of computer science principles in solving real-world logistical challenges but also provides a valuable testbed for developing strategies that could improve actual urban delivery operations.
